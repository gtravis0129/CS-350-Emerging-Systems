CS 350 portfolio for Emerging Systems Architectures and Technologies. Includes a Morse code LED/LCD messaging project and a thermostat prototype with sensor input, LCD output, LEDs, and serial status updates.
# CS 350 Portfolio

This repository contains my portfolio artifacts for CS 350: Emerging Systems Architectures and Technologies.

## Artifacts

- `Milestone3.py` – Morse code messaging project using LEDs, a button, and an LCD display.
- `Thermostat.py` – Thermostat prototype using sensor input, LEDs, an LCD, and serial communication.

## Project Summary

These projects demonstrate my ability to write interface software that controls hardware components and applies embedded systems concepts in practical applications.

## Reflection

This project helped me strengthen my skills in hardware and software integration, state-machine design, and writing code that is readable and adaptable for future projects.
